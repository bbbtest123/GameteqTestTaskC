Solution for the test task from GAMETEQ using C# and NUnit.

Run tests:
```
dotnet test
```

The task:
```
Описание:
Приложение находится по адресу http://test-task.gameteq.com/.
Оно предоставляет возможность просмотра, создания, изменения и удаления предложений
(Offers), а также позволяет следить за количеством имеющихся сущностей.
Задача: написать автотесты для веб-приложения, проверяющие функционал создания и
удаления предложений.
Нужно использовать:
1. Язык программирования C# или Java.
2. Фреймворки: Selenium WebDriver, NUnit/ MSTest / JUnit / TestNG.
3. Паттерн Page Object.
Нельзя использовать: высокоуровневые фреймворки на основе Selenium типа Selenide.
```
